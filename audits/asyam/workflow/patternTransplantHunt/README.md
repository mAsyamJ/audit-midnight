# Pattern Transplant Hunt

This workspace is a planning-only historical-pattern transplantation pass for the authorized local Morpho Midnight audit. It does not contain findings and it does not change protocol source or native tests.

## Workflow

1. Record public and local pattern sources without treating examples as proof.
2. Build a strict exclusion map from prior audits, PoC results, and closed queues.
3. Translate mechanisms into Midnight-specific hypotheses with exact surfaces and impact gates.
4. Select the top 25 local PoC candidates and detail the first five implementation plans.
5. Stop before PoC implementation and before writing Cantina findings.

## Evidence Discipline

- A dataset pattern is inspiration only.
- `new_poc_queue` is not a vulnerability classification.
- High or Medium requires a compiling Foundry PoC proving realistic non-dust harm to a non-consenting actor.
- Hashlock material remains hypothesis-only.
- Prior audited JSONs and existing findings are dedup sources.
